Mon Projet Web - Odin Recipes

Bienvenue dans mon projet de recettes réalisé dans le cadre du parcours *The Odin Project*. Ce projet a pour but de mettre en pratique les bases du développement web en créant un site simple et fonctionnel autour du thème de la cuisine.

Objectifs du projet

- Créer une structure HTML claire et bien organisée
- Utiliser les balises sémantiques pour améliorer l’accessibilité
- Apprendre à gérer les liens internes et les fichiers
- Développer une page d’accueil et des pages de recettes individuelles

Compétences démontrées

- Maîtrise du HTML de base
- Organisation de fichiers dans un projet web
- Utilisation de VS Code comme environnement de développement
- Gestion de version avec Git et GitHub

---

Ce projet marque le début de mon parcours en développement web. Il me permet de poser les fondations solides pour les étapes suivantes comme le CSS, JavaScript et bien plus encore.
