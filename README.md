My Web Project – Odin Recipes

Welcome to my recipe project created as part of the The Odin Project curriculum. The goal of this project is to apply the fundamentals of web development by building a simple and functional website centered around the theme of cooking.
 Project Objectives

    Create a clear and well-structured HTML layout

    Use semantic tags to improve accessibility

    Learn how to manage internal links and file organization

    Develop a homepage and individual recipe pages

🛠️ Skills Demonstrated

    Mastery of basic HTML

    File organization within a web project

    Use of VS Code as a development environment

    Version control with Git and GitHub

This project marks the beginning of my journey into web development. It lays a solid foundation for the next steps, such as CSS, JavaScript, and much more to come.
